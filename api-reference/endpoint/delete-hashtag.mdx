---
title: "Delete hashtag"
api: "DELETE /hashtags/{hashtag}"
description: "Delete a hashtag group by ID."
---

## Request

<ParamField path="hashtag" type="string" required>
  The hashtag group ID to delete.
</ParamField>

## Response

Returns `204 No Content` on success.

<RequestExample>
```bash
curl -X DELETE https://app.trypost.it/api/hashtags/e5f6a7b8-c9d0-1e2f-3a4b-5c6d7e8f9a0b \
  -H "Authorization: Bearer tp_your_api_key"
```
</RequestExample>
