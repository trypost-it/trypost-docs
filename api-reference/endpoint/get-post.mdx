---
title: "Get post"
api: "GET /posts/{post}"
description: "Get a specific post by ID with all its platform content, labels, and media."
---

## Request

<ParamField path="post" type="string" required>
  The post ID.
</ParamField>

## Response

Returns the full post details including platform-specific content, social accounts, and labels.

<RequestExample>
```bash
curl -X GET https://app.trypost.it/api/posts/9f1a2b3c-4d5e-6f7a-8b9c-0d1e2f3a4b5c \
  -H "Authorization: Bearer tp_your_api_key"
```
</RequestExample>
