---
title: "List hashtags"
api: "GET /hashtags"
description: "List all hashtag groups for the current workspace."
---

## Response

Returns all hashtag groups in the workspace.

<ResponseField name="data" type="array">
  <Expandable>
    <ResponseField name="id" type="string">
      The hashtag group ID.
    </ResponseField>
    <ResponseField name="name" type="string">
      The hashtag group name.
    </ResponseField>
    <ResponseField name="hashtags" type="string">
      The hashtags string.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      The creation timestamp.
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      The last update timestamp.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -X GET https://app.trypost.it/api/hashtags \
  -H "Authorization: Bearer tp_your_api_key"
```
</RequestExample>
