---
title: "List posts"
api: "GET /posts"
description: "List all posts for the current workspace with pagination."
---

## Request

<ParamField query="page" type="integer" optional>
  The page number for pagination. Defaults to `1`.
</ParamField>

## Response

Returns a paginated list of posts with their platforms, labels, and scheduling information.

<ResponseField name="data" type="array">
  <Expandable>
    <ResponseField name="id" type="string">
      The post ID.
    </ResponseField>
    <ResponseField name="status" type="string">
      The post status: `draft`, `scheduled`, `publishing`, `published`, `partially_published`, or `failed`.
    </ResponseField>
    <ResponseField name="synced" type="boolean">
      Whether the post content is synced across platforms.
    </ResponseField>
    <ResponseField name="scheduled_at" type="string">
      The scheduled date and time in `Y-m-d H:i:s` format.
    </ResponseField>
    <ResponseField name="published_at" type="string">
      The published date and time. Null if not yet published.
    </ResponseField>
    <ResponseField name="platforms" type="array">
      The post content for each connected social account.
      <Expandable>
        <ResponseField name="id" type="string">
          The post platform entry ID.
        </ResponseField>
        <ResponseField name="platform" type="string">
          The social platform name.
        </ResponseField>
        <ResponseField name="content" type="string">
          The post content for this platform.
        </ResponseField>
        <ResponseField name="content_type" type="string">
          The content type.
        </ResponseField>
        <ResponseField name="status" type="string">
          The publishing status for this platform.
        </ResponseField>
        <ResponseField name="enabled" type="boolean">
          Whether this platform entry is enabled.
        </ResponseField>
        <ResponseField name="social_account" type="object">
          The connected social account details.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="labels" type="array">
      The labels assigned to the post.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      The creation timestamp.
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      The last update timestamp.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -X GET https://app.trypost.it/api/posts \
  -H "Authorization: Bearer tp_your_api_key"
```
</RequestExample>
