---
title: "List social accounts"
api: "GET /social-accounts"
description: "List all connected social accounts for the current workspace."
---

## Response

Returns all connected social accounts sorted by platform.

<ResponseField name="data" type="array">
  <Expandable>
    <ResponseField name="id" type="string">
      The social account ID.
    </ResponseField>
    <ResponseField name="platform" type="string">
      The platform name: `linkedin`, `linkedin-page`, `x`, `facebook`, `instagram`, `tiktok`, `youtube`, `threads`, `pinterest`, `bluesky`, or `mastodon`.
    </ResponseField>
    <ResponseField name="display_name" type="string">
      The account's display name.
    </ResponseField>
    <ResponseField name="username" type="string">
      The account's username or handle.
    </ResponseField>
    <ResponseField name="is_active" type="boolean">
      Whether the account is active for posting.
    </ResponseField>
    <ResponseField name="status" type="string">
      The account connection status: `connected`, `disconnected`, or `token_expired`.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -X GET https://app.trypost.it/api/social-accounts \
  -H "Authorization: Bearer tp_your_api_key"
```
</RequestExample>
