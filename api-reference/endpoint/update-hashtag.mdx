---
title: "Update hashtag"
api: "PUT /hashtags/{hashtag}"
description: "Update a hashtag group's name or hashtags."
---

## Request

<ParamField path="hashtag" type="string" required>
  The hashtag group ID.
</ParamField>

<ParamField body="name" type="string" required>
  The new hashtag group name. Maximum 255 characters.
</ParamField>

<ParamField body="hashtags" type="string" required>
  The new hashtags string.
</ParamField>

## Response

Returns the updated hashtag group.

<RequestExample>
```bash
curl -X PUT https://app.trypost.it/api/hashtags/e5f6a7b8-c9d0-1e2f-3a4b-5c6d7e8f9a0b \
  -H "Authorization: Bearer tp_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Tech & AI",
    "hashtags": "#tech #ai #machinelearning #startup"
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "data": {
    "id": "e5f6a7b8-c9d0-1e2f-3a4b-5c6d7e8f9a0b",
    "name": "Tech & AI",
    "hashtags": "#tech #ai #machinelearning #startup",
    "created_at": "2025-01-10 08:00:00",
    "updated_at": "2025-01-10 09:00:00"
  }
}
```
</ResponseExample>
