## Supported media types

| Type | Description | Common formats |
|------|-------------|----------------|
| **Image** | Photos and graphics | JPG, PNG, GIF, WebP |
| **Video** | Video content | MP4, MOV |
| **Document** | PDFs for LinkedIn carousels | PDF |

## Upload limits

TryPost supports uploads up to **2 GB** per file, which covers even long-form video content. Large video files are uploaded using chunked uploads for reliability.

## Platform content types and media requirements

Each platform supports different content formats with specific media requirements:

### LinkedIn / LinkedIn Page

| Content type | Media | Max files | Aspect ratio | Notes |
|-------------|-------|-----------|-------------|-------|
| **Post** | Optional | 1 | — | Text with optional image or video |
| **Carousel** | Required | 20 | — | Multi-page PDF document |

### X (Twitter)

| Content type | Media | Max files | Notes |
|-------------|-------|-----------|-------|
| **Post** | Optional | 4 images or 1 video | Images and video cannot be mixed |

### Facebook

| Content type | Media | Max files | Aspect ratio | Notes |

## Media ordering

You can reorder media attachments by dragging them in the post editor. The order you set is the order they'll appear on the social platform (important for carousels and multi-image posts).

## File storage

Media files can be stored in different backends depending on your setup:

| Storage | Use case |
|---------|----------|
| **Local** | Development or small self-hosted instances |
| **AWS S3** | Production cloud storage |
| **Cloudflare R2** | Cost-effective S3-compatible storage |
| **Any S3-compatible** | MinIO, DigitalOcean Spaces, etc. |

<Note>
  On **TryPost Cloud**, media storage is handled automatically. Self-hosted users should configure cloud storage for production — see the [Configuration guide](/self-hosting/configuration).
</Note>

## FAQ

<AccordionGroup>
  <Accordion title="What's the maximum file size?">
    2 GB per file. Large files use chunked uploads for reliability.
  </Accordion>
  <Accordion title="Can I use the same media in multiple posts?">
    You can duplicate media attachments within the same post (e.g., to use the same image on different platforms). For different posts, you'll need to upload the media again.
  </Accordion>
  <Accordion title="What happens to media when I delete a post?">
    The media files are removed along with the post.
  </Accordion>
  <Accordion title="Are images compressed or resized?">
    TryPost uploads your media to social platforms as-is. Each platform applies its own compression and resizing rules.
  </Accordion>
</AccordionGroup>
