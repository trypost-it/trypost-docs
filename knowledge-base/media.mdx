## Supported media types

TryPost supports three types of media attachments:

| Type | Description | Common formats |
|------|-------------|----------------|
| **Image** | Photos and graphics | JPG, PNG, GIF, WebP |
| **Video** | Video content | MP4, MOV |
| **Document** | Documents and PDFs | PDF (LinkedIn only) |

## Platform-specific content types

Each platform supports different content formats. TryPost automatically maps your content to the right format:

### LinkedIn
- **Post** — Text with optional images or video
- **Carousel** — Multi-image document (PDF) posts

### LinkedIn Page
- **Post** — Same as personal LinkedIn posts
- **Carousel** — Same as personal LinkedIn carousels

### X (Twitter)
- **Post** — Text with optional images or video

### Facebook
- **Post** — Text with optional images or video
- **Reel** — Short-form vertical video
- **Story** — Temporary content (24 hours)

### Instagram
- **Feed** — Photo or video post in the main feed
- **Reel** — Short-form vertical video
- **Story** — Temporary content (24 hours)

### TikTok
- **Video** — Short or long-form video content

### YouTube
- **Short** — Short-form vertical video (YouTube Shorts)

### Threads
- **Post** — Text with optional images

### Pinterest
- **Pin** — Image pin
- **Video Pin** — Video pin
- **Carousel** — Multi-image pin

### Bluesky
- **Post** — Text with optional images

### Mastodon
- **Post** — Text with optional images or video

## File storage

Media files can be stored locally or in cloud storage (AWS S3, Cloudflare R2, or any S3-compatible service). See the [Configuration guide](/self-hosting/configuration) for storage setup.
