# Supported Platforms

TryPost supports 10 social media platforms:

| Platform | Content types |
|----------|--------------|
| [LinkedIn](/platforms/linkedin) | Posts, carousels (profile and company pages) |
| [X (Twitter)](/platforms/x-twitter) | Text posts with images or video |
| [Facebook](/platforms/facebook) | Posts, reels, stories |
| [Instagram](/platforms/instagram) | Feed posts, reels, stories |
| [TikTok](/platforms/tiktok) | Short-form video |
| [YouTube](/platforms/youtube) | YouTube Shorts |
| [Threads](/platforms/threads) | Text posts with images |
| [Pinterest](/platforms/pinterest) | Pins, video pins, carousels |
| [Bluesky](/platforms/bluesky) | Text posts with images |
| [Mastodon](/platforms/mastodon) | Text posts with media (any instance) |

## Connecting accounts

On **TryPost Cloud**, simply go to **Accounts** in the dashboard and click the platform you want to connect. You'll be redirected to authorize TryPost.

<Note>
  **Self-hosting?** You'll need to create developer apps and configure API credentials for each platform. See the [Self-Hosting configuration guide](/self-hosting/configuration) for details.
</Note>
